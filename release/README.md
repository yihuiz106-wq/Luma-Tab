# Luma Tab Release

Included snapshots: `1.0.0`, `1.0.1`, `1.0.2`

## Install

1. Choose one of the bundled release folders such as `Luma-Tab-v1.0.2/`
2. Open `chrome://extensions/`
3. Enable `Developer mode`
4. Click `Load unpacked`
5. Select that folder

There is also a bundled archive: `Luma-Tab-v1.0.2-unpacked.zip`

If you want to verify or customize the build yourself, build from the project root and load the generated output directory.

## Notes

- This build replaces Chrome's new tab page.
- DeepSeek features require your own API key.
- The source tree may be ahead of the bundled release snapshots.
- If you want to preserve your settings and categories across reinstalls, use the in-app `Export Data` / `Import Data` tools in `Settings`.
